# QuickDrop Plans

These features are not part of the current extension yet, but are planned for upcoming versions.

## Planned Features

- **Global Search**: Search saved content by title, URL, note, group, tag, or selected text.
- **Star / Important**: Mark important items so users can quickly revisit them later.
- **Duplicate Detection**: Detect when the same URL is already saved and show an "Already saved" state instead of creating duplicates.
- **Archive**: Move old items out of the active list without deleting them permanently.
- **Import Bookmarks**: Import existing Chrome bookmarks into QuickDrop.
- **Share Group**: Share a group of saved resources with another user, such as project docs or research links.
- **Subgroups**: Create smaller sections inside groups so users can organize saved resources by topic, project phase, or priority.
- **Reminders and Notifications**: Let users set reminders for saved items and receive browser push notifications when it is time to revisit them.
- **QuickDrop App**: Build a standalone app experience for QuickDrop so users can manage saved resources outside the browser extension.
- **AI Summary**: Later, summarize long pages or docs and save the summary with the original item.
